Workflow:
  1. User taps a time cell (e.g., "0:45") — row highlights
  2. User sets the current wall clock time (e.g., 08:45)
  3. User taps "Sync Time" — the table anchors "0:45" to 08:45
  4. Past rows dim (0:00, 0:15, 0:30 become translucent)
  5. "Now" row gets a blue highlight + "◀ (now)" label
  6. Future rows remain full opacity
  7. Progress bar at top shows elapsed fraction
  8. User can "Clear Sync" to reset
